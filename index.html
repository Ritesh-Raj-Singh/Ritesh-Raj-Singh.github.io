Featured Projects

ThreadsafeQueueLib

High-performance Lock-Free & Wait-Free Queue Library

Developed a high-performance C++ concurrency library featuring wait-free SPSC and lock-free MPSC/MPMC queues. By architecting bounded ring-buffers and chunk-based linked nodes, and carefully managing atomic memory ordering and cache-line alignment, the library effectively eliminates false sharing. It achieved an impressive 121.3M Ops/s (SPSC) and 22.8M Ops/s (MPMC) in benchmarks.

C++ Concurrency Lock-Free

RISC-V Hardware Accelerator

Real-Time Matrix Multiplication

Implemented a 5-stage pipelined 32-bit RISC-V (RV32IMF) processor on an FPGA using Verilog, integrated with a 4x4 Systolic Array via MMIO to accelerate real-time image filtering. I heavily optimized the CPU pipeline using data forwarding and active load-use hazard detection, and built a dynamic branch predictor utilizing a 64-entry BHT and BTB. The accelerator utilizes Ping/Pong double buffering to hide memory latency and streams data over a custom UART protocol.

Verilog FPGA RISC-V C++

VELORA
